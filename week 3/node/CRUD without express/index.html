CRUD without Express.js

Create new user:

http://localhost:3000/create

Retrieve data:

http://localhost:3000/retrieve

Sample query:
{"103": {
"name": "prravan",
"age": "202"
}
}

Update existing user:

http://localhost:3000/update

Sample query:
{"103": {
"name": "prravan",
"age": "2022"
}
}

Delete existing user:

http://localhost:3000/delete

Sample query:
{"103": ""}
